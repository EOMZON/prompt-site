# Prompt Site

Independent prompt-only frontend for `prompt.zondev.top`.

## What It Is

- prompt browsing
- prompt detail pages
- copy-first prompt reading
- GitHub-backed source links

## What It Is Not

- not the skills site
- not a migration layer into `skills.zondev.top`
- not a mixed prompt + skills frontend

## Local Development

```bash
npm run build
```

Vercel publishes the generated `dist/` directory.
